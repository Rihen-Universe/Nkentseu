## 1. Vue d'ensemble du Launcher

### Description Générale

Le Launcher est la **fenêtre de démarrage** de l'application d'animation, affichée avant l'ouverture de l'éditeur 3D complet. C'est une fenêtre indépendante entièrement custom — pas de chrome système, pas de dialog natif. Tout est dessiné par l'application elle-même via NKCanvas.

L'identité visuelle est **cinématique et professionnelle** : fond sombre profond, visuels d'animation en arrière-plan animés subtilement, typographie large, accents couleur or/ambre pour distinguer du Launcher IDE (cyan).

### Dimensions & Comportement

- **Taille par défaut** : 1100 × 680 px, centrée à l'écran
- **Redimensionnable** : oui, minimum 820 × 520 px
- **Frameless** : barre de titre entièrement custom, boutons Fermer/Réduire/Maximiser dessinés
- **Drag** : glisser la barre de titre pour déplacer
- **Double clic** barre de titre → maximiser / restaurer
- **Fond** : image ou vidéo d'animation en boucle très subtile en arrière-plan (opacité 8%), ou dégradé animé

### Groupes des Récents

Les projets sont groupés par date, chaque groupe est pliable :

- **⭐ Épinglés** — toujours en premier, non groupés par date
- **Aujourd'hui**
- **Cette semaine**
- **Ce mois**
- **Plus anciens**

### Barre de Filtre

```
🔍 [Filtrer les projets récents...]    [🎬 Film ▼]  [🎮 Jeu ▼]  [✅ Rendu ▼]  [Trier: Date ▼]
```

Filtres disponibles :
- **Par nom** : filtre texte temps réel
- **Par type** : Film, Jeu, Previz, Publicité, Custom
- **Par statut** : En cours, Rendu terminé, Export terminé, Erreur IA
- **Par moteur** : Cycles X, EEVEE Next, Export uniquement
- **Tri** : Date de modification, Nom, Taille, Score IA

### Panneau Gauche — Favoris & Appareils

- **Favoris** : dossiers fréquents, drag d'un dossier depuis la zone principale pour ajouter
- **Clic droit sur favori** → Renommer, Supprimer des favoris, Ouvrir dans le terminal
- **Appareils** : disques locaux, réseau, USB, Cloud monté
- Clic sur un item → naviguer vers ce dossier

### Zone Principale — Colonnes

| Colonne | Description |
|---------|-------------|
| Icône + Nom | 📁 dossier · 🎬 Film · 🎮 Jeu · 🎥 Previz · 📦 archivé |

## 16. Flux de Travail Complets

### Workflow 1 — Nouveau Projet Film depuis Zéro

```
1. Lancer AnimStudio Pro → Launcher s'ouvre
2. Clic "➕ Nouveau Projet"  ou  Ctrl+N
3. Étape 1: Sélectionner "🎬 Film" · Nom: "Combat_Hero" · FPS: 24 · 250fr
4. Étape 2: Environnement HDRI · Cycles X · Physique activée · Caméra cinéma
5. Étape 3: Ajouter "Mannequin Humanoid" · Cloth + Hair · T-Pose + A-Pose
6. Étape 4: Vérifier le résumé → "✅ Créer le Projet"
7. Animation de création (~5s)
8. Écran de chargement → Éditeur 3D s'ouvre avec le projet vide
```

### Workflow 2 — Reprendre un Projet Récent

```
1. Lancer AnimStudio Pro → Launcher s'ouvre
2. Voir "Combat_Hero_v3" dans les récents → Clic simple
3. Écran de chargement (session précédente restaurée : frame 145)
4. Éditeur s'ouvre directement au dernier point de travail
```

### Workflow 3 — Générer une Animation avec l'IA

```
1. Launcher → "🤖 Générer avec l'IA"  ou  Ctrl+G
2. Saisir la description: "Un guerrier qui saute et frappe"
3. Choisir le rig: Mannequin Humanoid · FPS: 24 · Qualité: Haute
4. Clic "🤖 Générer l'animation"
5. Écran de progression (~30-60 secondes selon GPU)
6. Panneau de sélection des 4 variantes avec prévisualisations
7. Sélectionner la variante 1 (score: 91%)
8. Clic "✅ Appliquer et Ouvrir dans l'éditeur"
9. Écran de chargement → Éditeur s'ouvre avec l'animation générée
```

### Workflow 4 — Importer une Capture de Mouvement

```
1. Launcher → "📂 Ouvrir un Projet"  ou  Ctrl+O
2. File Browser custom → naviguer vers le projet existant
3. Double clic sur "MonProjet.asproj" → validation
```
